# CJ-DEVOPS

Ce dépôt regroupe l’ensemble de mes travaux DevOps : playbooks Ansible, TPs Docker, labs Vagrant, applications, et documentation.

## Structure du projet

- **branches/** : Copie locale de chaque branche Git (voir branches/README.md pour le détail)
- **ansible-automation/** : Playbooks et rôles Ansible pour l’infrastructure
- **docker/** : TPs Docker, exemples, exercices, best practices
- **test/** : Labs et environnements de test (machines-test)
- **uyoop-app/** : Application Uyoop (PHP et version dockerisée)
- **vagrant-vms/** : TPs Vagrant, machines virtuelles

## Utilisation

- Naviguez dans chaque dossier pour explorer les TPs, playbooks, apps ou labs
- Ajoutez vos nouveaux cours ou exercices dans la branche ou le dossier approprié
- Consultez la documentation présente dans chaque dossier clé

## Objectif

Ce repo est conçu pour :
- Valoriser mes compétences DevOps
- Servir de support pédagogique et portfolio
- Faciliter la montée en compétences et la présentation professionnelle
